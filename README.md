## Technologies utilisées 👨‍💻

- Ubuntu > Terminal 
- Visual Studio Code
- Git
- HTML
- CSS

## Comment utiliser ⏯️

a . Clonez ce référentiel
    ```bash
    git clone https://github.com/kferrerux/christmas-even-website
    ```

b . Ouvrez le fichier `index.html` dans votre navigateur

c . Explorez et appréciez!
